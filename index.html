Welcome to HCI Lab 01

This website is developed for Lab & Assignment 01 to practice semantic HTML while comparing Don Norman’s Action Cycle, GOMS, and Keystroke-Level Model (KLM) based on Dix et al., Human–Computer Interaction (Chapters 9 & 12).

Lab Scope

  • Understand Norman’s Action Cycle & the gulfs of execution and evaluation
  • Compare task analysis using GOMS: goals, operators, methods, selection rules
  • Analyze task execution using KLM operators and timing predictions

Page Overview

  • Home: Main landing page
  • About: Don Norman biography and GOMS creators
  • CV: Student information and HCI skills
  • Contact: Feedback form
  • Interaction Analysis: Real-world task examples
  • Norman–GOMS–KLM: Comparison tables and models
  • Who Paid Task: Interactive task example
  • UI Evaluation: Bad vs Good UI Design Examples
  • HTML Errors: Common mistakes and corrections
